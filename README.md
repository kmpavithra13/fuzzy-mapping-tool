**FuzzyMap — Data Standardization Tool**

FuzzyMap helps you clean messy data by matching incorrect values with standardized ones.

For example:

| Messy value | Standard value |
|-------------|---------------|
| U.P | Uttar Pradesh |
| Uttar prades | Uttar Pradesh |
| Pradesh, Uttar | Uttar Pradesh |

The tool uses fuzzy matching to automatically suggest the correct values.

Everything runs in your browser — your data is not uploaded anywhere.How the tool works

The cleaning process happens in five simple steps:

* Upload your dataset
* Upload a reference dictionary (correct values)
* Select the column you want to clean
* Review suggested matches
* Download the cleaned dataset

Typical Use Cases: This tool is useful for Data journalism, Research datasets, Government datasets, Cleaning survey data and Standardizing organization names
